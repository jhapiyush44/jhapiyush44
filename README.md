**Tech Stack:** `FastAPI` `LLMs` `Embeddings` `Streamlit` `SentenceTransformers`

An AI-powered recruiter tool that automates resume parsing, scoring, and candidate ranking using hybrid ML approaches.

- 🎯 **Hybrid Scoring:** 50% rule-based + 30% semantic embeddings + 20% experience matching
- ⚡ **Performance:** 70% reduction in API costs via intelligent caching
- 📊 **Impact:** 80% reduction in manual resume review time
- 🧠 **Features:** LLM-based interest simulation, top-K ranking, interactive dashboard

**Key Learnings:** Production LLM integration, cost optimization, semantic search systems

- 📈 **Scale:** 3.9M rows, 40+ engineered features (lag, rolling stats, temporal encoding)
- 🎯 **Performance:**

  - Regression: R² = 0.92, MAE = 12.3
- 🐳 **Deployment:** Dockerized FastAPI service with batch predictions & model versioning
- 💾 **Optimization:** 80% storage reduction using Parquet columnar format

**Key Learnings:** Feature engineering at scale, model deployment, API design
